نسخة GitHub Pages الجاهزة

1) ارفع جميع محتويات هذا المجلد إلى جذر المستودع، ويجب أن يكون index.html ظاهرًا في الصفحة الأولى للمستودع.
2) افتح Settings ثم Pages.
3) اختر Deploy from a branch.
4) اختر main و /(root) ثم Save.
5) انتظر اكتمال النشر، ثم افتح الرابط الذي يظهر داخل صفحة Pages.

مهم: صفحة المستودع github.com تعرض الملفات أو README، أما التطبيق فيظهر على رابط github.io الموجود في Settings > Pages.
