# رحلة العربية

تطبيق تعليمي تفاعلي لمنهج اللغة العربية للصف الأول الإعدادي – الفصل الدراسي الأول، بإشراف الأستاذ محمد سعيد جعباص.

## المزايا

- 3 وحدات تعليمية و12 درسًا مرتبة وفق المنهج.
- 30 سؤالًا مباشرًا لكل درس، وبنك من 50 سؤالًا لكل وحدة.
- فتح متتابع للدروس والوحدات بعد اجتياز الاختبار بنسبة 70%.
- حفظ تلقائي لبيانات الطالب، آخر درس، وآخر سؤال على الجهاز.
- أغلفة ثلاثية الأبعاد مخصصة للوحدات والدروس.
- نافذة تعريف احترافية بالمعلم، مع صورة ومؤهلات مستخلصة من سيرته الذاتية.
- تصميم عربي RTL متجاوب مع الهاتف والكمبيوتر وخط Alexandria المضمّن.

## التشغيل محليًا

يتطلب المشروع Node.js 22.13 أو أحدث.

```bash
npm install
npm run dev
```

ثم افتح `http://localhost:3000`.

## التحقق وبناء نسخة الإنتاج

```bash
npm test
npm run build
npm run start
```

## رفع المشروع إلى GitHub

أنشئ مستودعًا جديدًا، ثم ارفع محتويات هذا المجلد. مجلدات البناء والحزم المؤقتة مستبعدة تلقائيًا بواسطة `.gitignore`، ويعمل فحص البناء تلقائيًا عبر GitHub Actions عند كل رفع.

## بنية المحتوى

- `app/CurriculumApp.tsx`: الشاشات، الحفظ، القفل، والاختبارات.
- `app/curriculum-data.ts`: محتوى الوحدات والدروس والأسئلة.
- `app/globals.css`: الهوية البصرية والتجاوب.
- `public/images`: جميع أغلفة الوحدات والدروس وصورة المعلم.
